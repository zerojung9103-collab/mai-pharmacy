# 🔔 วิธีเปิด "แจ้งเตือนเด้ง" (Push Notifications) — ทำครั้งเดียวจบ

ระบบแจ้งเตือนเด้ง = ข้อความเด้งเข้ามือถือ/คอมของทุกคน **แม้ไม่ได้เปิดแอปอยู่**
เช่น เตือนก่อนเวร 30 นาที · หมดเวรยังไม่ปิดร้าน · สลิปเงินเดือนออก · เงินปิดร้านไม่ตรง (ถึงผู้ดูแล) ฯลฯ

ตัวส่งอยู่ใน Cloudflare Worker ตัวเดียวกับที่รันเว็บอยู่แล้ว — **ฟรี ไม่ต้องสมัครอะไรเพิ่ม**
แต่ต้อง "วางกุญแจ 2 ตัว" ให้ระบบก่อน (ทำครั้งเดียว ใช้ได้ตลอด):

| กุญแจ | เอามาจาก | เอาไปวางที่ |
|---|---|---|
| ① กุญแจ Web Push (VAPID) | Firebase Console | ในแอป (โมดัล 🔔) |
| ② กุญแจ Service Account (ไฟล์ JSON) | Firebase Console | Cloudflare (Secret) |

---

## ขั้นที่ 1 — เอากุญแจ Web Push จาก Firebase มาวางในแอป

1. เปิด https://console.firebase.google.com → เลือกโปรเจกต์ **naimaphat-pharmacy**
2. กดฟันเฟือง ⚙️ (มุมซ้ายบน) → **Project settings**
3. ไปแท็บ **Cloud Messaging**
4. เลื่อนลงหาหัวข้อ **Web configuration → Web Push certificates**
5. ถ้ายังไม่มีกุญแจ ให้กด **Generate key pair** — จะได้ตัวอักษรยาวๆ ขึ้นต้นด้วย `B`
6. กดปุ่มคัดลอก 📋 ข้างกุญแจนั้น
7. เปิดแอปร้าน (ล็อกอินเป็นผู้ดูแล) → **บัญชีของฉัน → 🔔 การแจ้งเตือนเด้ง**
8. จะเห็นกล่องเหลือง "ติดตั้งครั้งแรก" → วางกุญแจในช่อง แล้วกด **บันทึกกุญแจ**

## ขั้นที่ 2 — เอากุญแจ Service Account ไปวางใน Cloudflare

**2.1 ดาวน์โหลดไฟล์กุญแจจาก Firebase:**
1. ยังอยู่ใน Project settings → ไปแท็บ **Service accounts**
2. กดปุ่ม **Generate new private key** → ยืนยัน → ได้ไฟล์ `.json` ดาวน์โหลดมา 1 ไฟล์
3. เปิดไฟล์นั้นด้วย Notepad (คลิกขวา → Open with → Notepad) แล้ว **คัดลอกทั้งหมด** (Ctrl+A แล้ว Ctrl+C)
   ⚠️ ไฟล์นี้คือกุญแจบ้าน — อย่าส่งให้ใคร อย่าโพสต์ลงไลน์/GitHub

**2.2 วางใน Cloudflare:**
1. เปิด https://dash.cloudflare.com → เมนูซ้าย **Workers & Pages** → กดที่ **mai-pharmacy**
2. ไปแท็บ **Settings** → หัวข้อ **Variables and Secrets**
3. กด **Add** →
   - Type: เลือก **Secret**
   - Variable name: พิมพ์ `FIREBASE_SERVICE_ACCOUNT` (ตัวใหญ่เป๊ะๆ แบบนี้)
   - Value: **วางเนื้อไฟล์ JSON ทั้งหมด** ที่คัดลอกมา
4. กด **Deploy** (หรือ Save)

## ขั้นที่ 3 — อัปโหลดไฟล์เวอร์ชันใหม่ขึ้น GitHub (ถ้ายังไม่ได้ทำ)

ไฟล์ที่เปลี่ยนในรอบนี้: `app.html` · `sw.js` · `worker-entry.js` · `wrangler.toml`
อัปขึ้น GitHub ตามปกติ → Cloudflare จะ deploy เองอัตโนมัติ
(ตัวปลุกทุก 10 นาทีจะตั้งให้เองจากไฟล์ wrangler.toml — ดูได้ที่แท็บ Settings → Triggers → Cron Triggers)

## ขั้นที่ 4 — ทดสอบ

1. เปิดแอป → **บัญชีของฉัน → 🔔 การแจ้งเตือนเด้ง**
2. กด **🔔 เปิดรับที่เครื่องนี้** → เครื่องถามสิทธิ์ → กด "อนุญาต"
3. กด **📨 ทดสอบส่งหาฉัน** → ควรมีแจ้งเตือนเด้งภายในไม่กี่วินาที (ลองพับแอปไว้ก่อนก็ได้)
4. ให้พนักงานทุกคนทำข้อ 1–2 ที่เครื่องของตัวเอง (มีแบนเนอร์เขียวชวนเปิดในกระดิ่ง 🔔 ของแอปอยู่แล้ว)

## 📱 ข้อควรรู้รายเครื่อง

- **Android**: ใช้ได้ทั้งจาก Chrome และแอปที่ติดตั้งลงเครื่อง — แนะนำติดตั้งแอปลงหน้าจอ
- **iPhone/iPad**: ⚠️ **ต้องติดตั้งแอปลงหน้าจอโฮมก่อน** (Safari → ปุ่มแชร์ → "เพิ่มลงไปยังหน้าจอโฮม") แล้วเปิดจากไอคอนนั้น ถึงจะกดเปิดรับแจ้งเตือนได้ (ต้อง iOS 16.4 ขึ้นไป)
- **คอม (Chrome/Edge)**: ใช้ได้เลย เด้งมุมจอ
- เครื่องหนึ่งกดเปิดรับได้หลายเครื่อง (มือถือ+คอม) — เด้งทุกเครื่องที่เปิดไว้

## 🎛️ ตั้งค่าได้อะไรบ้าง (รายคน — แต่ละคนตั้งของตัวเอง)

- เปิด/ปิดรายหัวข้อ (14 หัวข้อ · หัวข้อผู้ดูแลเห็นเฉพาะผู้ดูแล)
- ช่วงห้ามกวน (ค่าเริ่มต้น 22:00–07:00) — เรื่องด่วน (เงินปิดร้านไม่ตรง / สาขายังไม่ปิดร้าน) เด้งเสมอ
- ทั้งหมดอยู่ใน บัญชีของฉัน → 🔔

## 🔧 ถ้าไม่เด้ง เช็คตามนี้

1. กด "ทดสอบส่งหาฉัน" แล้วขึ้น "ตัวส่งยังไม่พร้อม" → ยังไม่ได้ทำขั้นที่ 2 (Secret ใน Cloudflare) หรือชื่อ Secret พิมพ์ไม่ตรง
2. ขึ้น "ยังไม่มีเครื่องลงทะเบียน" → กด "เปิดรับที่เครื่องนี้" ก่อน
3. iPhone กดเปิดรับไม่ได้ → ยังไม่ได้เปิดจากไอคอนหน้าจอโฮม
4. เคยกด "บล็อก" ตอนเครื่องถามสิทธิ์ → ไปเปิดในตั้งค่าเบราว์เซอร์/ตั้งค่าเครื่อง → การแจ้งเตือน
5. แจ้งเตือนตามเวลา (ก่อนเวร/บรีฟค่ำ) ไม่มา → เช็ค Cloudflare → mai-pharmacy → Settings → Triggers ว่ามี Cron `*/10 * * * *` อยู่
