# 🔔 วิธีเปิดการแจ้งเตือนเด้ง (ทำครั้งเดียว)

ระบบแจ้งเตือนต้องใช้ "กุญแจ" 2 ดอกจาก Firebase — ทำตามนี้ทีละขั้น ใช้เวลา ~10 นาที

---

## ขั้นที่ 1 — อัปโหลดไฟล์ขึ้น GitHub (เหมือนทุกครั้ง + มีโฟลเดอร์ใหม่)

ลากไฟล์เหล่านี้ขึ้น GitHub:
- `app.html` และ `sw.js` (เหมือนทุกครั้ง)
- `netlify.toml` (ไฟล์ตั้งค่า — ทับของเดิม)
- **โฟลเดอร์ `netlify` ทั้งโฟลเดอร์** (ข้างในมีตัวส่งแจ้งเตือน 2 ตัว)
  - ถ้าลากทั้งโฟลเดอร์ไม่ได้ ให้กด Add file → Upload files แล้วลากโฟลเดอร์ `netlify` ลงไปทั้งก้อน GitHub จะเก็บโครงสร้างให้เอง

## ขั้นที่ 2 — วางกฎ Firestore ใหม่

เอาเนื้อหาไฟล์ `firestore-rules-ใหม่.txt` (ตัวล่าสุด) ไปวางที่
**Firebase Console → Firestore Database → Rules → วางทับ → Publish**

## ขั้นที่ 3 — กุญแจดอกที่ 1: กุญแจ Web Push (วางในแอป)

1. เปิด [Firebase Console](https://console.firebase.google.com) → โปรเจกต์ **naimaphat-pharmacy**
2. เฟือง ⚙️ มุมซ้ายบน → **Project settings** → แท็บ **Cloud Messaging**
3. หัวข้อ **Web configuration → Web Push certificates** → กด **Generate key pair**
4. คัดลอกกุญแจที่ได้ (ตัวอักษรยาวๆ ขึ้นต้นด้วย B)
5. เปิดแอปใหม่เภสัช → รูปโปรไฟล์ → **🔔 การแจ้งเตือนเด้ง** → วางกุญแจในช่อง → กด **บันทึกกุญแจ**

## ขั้นที่ 4 — กุญแจดอกที่ 2: กุญแจเซิร์ฟเวอร์ (วางใน Netlify)

1. Firebase Console → ⚙️ **Project settings** → แท็บ **Service accounts**
2. กด **Generate new private key** → ยืนยัน → จะได้ไฟล์ `.json` ดาวน์โหลดมา
   ⚠️ ไฟล์นี้คือกุญแจเข้าถึงข้อมูลทั้งหมด — **ห้ามส่งให้ใคร ห้ามอัปขึ้น GitHub เด็ดขาด**
3. เปิดไฟล์นั้นด้วย Notepad/TextEdit แล้วคัดลอกเนื้อหา**ทั้งหมด**
4. เปิด [Netlify](https://app.netlify.com) → เว็บของร้าน → **Site configuration → Environment variables**
5. กด **Add a variable**:
   - Key: `FIREBASE_SERVICE_ACCOUNT`
   - Value: วางเนื้อหาไฟล์ JSON ทั้งก้อน
   - กด **Create variable**
6. ไปแท็บ **Deploys** → กด **Trigger deploy → Deploy site** (ให้ตัวส่งรู้จักกุญแจใหม่)

## ขั้นที่ 5 — เปิดรับที่เครื่องของแต่ละคน

ทุกคน (รวมคุณ) ทำที่เครื่องตัวเอง:
1. เปิดแอปจาก**ไอคอนหน้าจอโฮม** (สำคัญมากสำหรับ iPhone)
2. จะมีแถบ 🔔 ชวนเปิดบนหน้าแรก กด **เปิดเลย** → กด **อนุญาต**
   (หรือเข้า รูปโปรไฟล์ → 🔔 การแจ้งเตือนเด้ง → เปิดรับที่เครื่องนี้)
3. กด **📨 ทดสอบส่งหาฉัน** — ถ้าเด้ง = เสร็จเรียบร้อย 🎉
